# 概览

考虑到性能和便利，hstream不会默认开启安全功能特性（如加密、认证等），但如果客户端连接服务端的网络是不可信的，那么应该启用这个功能。

hstream已经支持的安全特性：

  • 数据加密：可以避免客户端和服务端之间传输的数据被中间人监听和篡改。
  • 认证：为服务端提供认证客户端合法性的机制，并给授权功能提供统一的接口。
